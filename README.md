## Introduction
The program connects to SonarQube server and get the vulnerabilities and generate the word documents.

## How to build

### Windows 
```shell
gradlew build 
```

### Linux
```shell
./gradlew build 
```

## How to run

### Windows
```shell
gradlew run 
```

### Linux
```shell
./gradlew run 
```

## Output file
You can find the result in "output" folder
